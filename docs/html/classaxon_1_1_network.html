◆ input() [2/2]

void axon::Network::input ( const std::vector< std::vector< double > > & input)

Inputs values into the network and changes internal state to reflect this.

This only stores the values, it does not compute anything. It you call it again before calculating it will fail and keep the previous input. This version takes multiple inputs to compute in parallel.
